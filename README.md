# Mdisk Search Bot

[![Open Source Love svg2](https://badges.frapsoft.com/os/v2/open-source.svg?v=103)](https://github.com/8769ANURAG/mdisk-search-bot)   


We have to use Bot for Inline Search & Userbot for Searching in Channels. So both Bot & Userbot will work together.

- [x] Auto Filter
- [x] Broadcast 
- [x] Spelling Check Feature
